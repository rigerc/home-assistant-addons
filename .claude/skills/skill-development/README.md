# skill-development

Tools for creating, auditing, and maintaining Claude Code skills.

## Keywords

create skill, new skill, scaffold skill, skill template, audit skill, review skill, check skill, validate skill, skill quality, skill maintenance, fork skills, skill development, build skill, write skill, skill author, skill creator, skill standards, skill compliance

## Commands

- `/create-skill <name>` - Scaffold new skill from template
- `/review-skill <name>` - Quality review existing skill
- `/audit` - Multi-agent parallel audit
- `/audit-skill-deep <name>` - Deep content validation
- `/deep-audit <name>` - Extended audit with doc scraping

## Templates

Located in `templates/` subdirectory:
- `skill-skeleton/` - Complete directory to copy
- `SKILL-TEMPLATE.md` - SKILL.md starter
- `README-TEMPLATE.md` - README.md starter

## For Forkers

Install this skill to get the same development tooling as the main repo:

```bash
/plugin install ./skills/skill-development
```
