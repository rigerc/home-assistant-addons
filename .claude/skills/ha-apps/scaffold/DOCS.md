# Example Add-on Documentation

## Overview

This add-on provides a basic scaffold for building Home Assistant add-ons.

## Installation

1. Navigate to the Supervisor panel in Home Assistant
2. Click on the Add-on Store tab
3. Find "Example Add-on" in the list
4. Click on the add-on and then click "Install"

## Configuration

This add-on currently has no configuration options.

## Usage

After starting the add-on, it will run in the background.
