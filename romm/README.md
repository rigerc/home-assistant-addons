# Romm

![Upstream Status](https://img.shields.io/endpoint?url=https://raw.githubusercontent.com/rigerc/home-assistant-addons/main/.versions/romm.json) ![Add-on Version](https://img.shields.io/badge/version-1.0-blue)

Self-hosted ROM collection manager and emulator launcher.
Scan, organize, and manage game collections across 400+ platforms with
automatic metadata fetching and in-browser gameplay.


## About

Home Assistant add-on for [Romm](https://github.com/rommapp/romm).

## Installation

1. In Home Assistant, go to **Settings** → **Add-ons** → **Add-on Store**
2. Click the three dots (⋮) in the top right corner
3. Select "Add repository from URL"
4. Enter: `https://github.com/rigerc/home-assistant-addons`
5. Click "Add"
6. Find "Romm" in the list and click it
7. Click "Install"

## Supported Architectures

Our add-on supports the following architectures with manifests.

- `aarch64`
- `amd64`


## Configuration

The add-on can be configured via the Home Assistant UI. See the upstream project for configuration details.
