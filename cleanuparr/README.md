# Cleanuparr

![Latest Upstream](https://img.shields.io/github/release/cleanuparr/cleanuparr?label=latest) ![Add-on Version](https://img.shields.io/badge/version-0.2.11-blue) ![Upstream Version](https://img.shields.io/badge/upstream-2.5.1-informational)

Automated cleanup tool for Sonarr, Radarr, and download clients.
Removes unwanted or blocked files, manages stalled downloads, and
enforces blacklists/whitelists. Includes malware detection and
automatic search triggers for removed content.


## About

Home Assistant add-on for [Cleanuparr](https://github.com/Cleanuparr/Cleanuparr).

## Installation

1. In Home Assistant, go to **Settings** → **Add-ons** → **Add-on Store**
2. Click the three dots (⋮) in the top right corner
3. Select "Add repository from URL"
4. Enter: `https://github.com/rigerc/home-assistant-addons`
5. Click "Add"
6. Find "Cleanuparr" in the list and click it
7. Click "Install"

## Supported Architectures

Our add-on supports the following architectures with manifests.

- `aarch64`
- `amd64`


## Configuration

The add-on can be configured via the Home Assistant UI. See the upstream project for configuration details.
