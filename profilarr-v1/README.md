Profile management for *arr applications (Sonarr, Radarr, etc.)

## About

Home Assistant add-on for [Profilarr v1](https://github.com/Dictionarry-Hub/profilarr).

## Installation

1. In Home Assistant, go to **Settings** → **Add-ons** → **Add-on Store**
2. Click the three dots (⋮) in the top right corner
3. Select "Add repository from URL"
4. Enter: `https://github.com/rigerc/home-assistant-addons`
5. Click "Add"
6. Find "Profilarr v1" in the list and click it
7. Click "Install"

## Supported Architectures

Our add-on supports the following architectures with manifests.

- `amd64`
- `aarch64`

The add-on can be configured via the Home Assistant UI. See the upstream project for configuration details.
