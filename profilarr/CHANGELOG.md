## [0.1.0] - 2026-01-31

### Added

- Initial release of Profilarr add-on
- Support for amd64 and aarch64 architectures
- Ingress support for embedded UI access in Home Assistant
- Direct port access (6868) for API clients
- Configurable authentication modes (on, local, oidc, off)
- Integration with Home Assistant timezone settings
- Based on upstream Profilarr v1.1.3
- Git-backed configuration storage support
- Quality profile management for Radarr and Sonarr
- Custom format management
- Release profile management
